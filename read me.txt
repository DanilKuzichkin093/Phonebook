ЦЕЛЬ:

ГРУППА 2.

ТЕМА ПРОЕКТА: Телефонная книга Сервис

Идея о том, что пользователь может использовать сервис для сохранения и поиска
контактов, ТРИ РАЗДЕЛЬНЫХ ЗАДАЧИ:

Задача 1.
Создайте файл XML для произвольной передачи данных, файл схемы для этого файла и
файл (ы) XSL для
различных преобразований (предпочтительно, по крайней мере, для
отображения данных в HTML и) 0VIL)

XML файл должен содержать идентификатор, имя, фамилию, телефон, адрес электронной
почты могут быть добавлены к вашей собственности.

XML должен иметь 2 или 3 логических измерения.

<dim1>
  <dim2>
    <dim3>
    </dim3>
  </dim2>
</dim1>


Отображение данных в виде таблицы HTML с использованием файлов XSLT и PHP с
различными функциями (например, поиск контактных данных по имени). Придумайте хотя бы
3 функции.

Задача 2. 
Веб-сервис и клиентское приложение.

Создайте веб-сервис и клиентское приложение, которое позволит предоставлять услуги,
различать пользователей
Сервис e-eBay, который будет создан:

• Должен быть создан с использованием технологии ASP.NET MVC Web API.

• База данных должна содержать не менее 3 таблиц.

• должен предоставить услугу медсестре

• должен поддерживать многопользовательские возможности (администратор,

зарегистрированный пользователь, аноним).

Для создания службы телефонной книги и клиентского приложения:

• Пользователь может добавлять и удалять контакты в сервисе

• Пользователь может выполнять поиск по имени контакта / номеру телефона и т. Д.

• Пользователи могут быть добавлены в группы

• Пользователь может добавить требуемые типы контактных данных.

• Пользователи могут делиться контактами и группами контактов

Задание 3. Проектная документация.

Этапы создания проекта с соответствующими описаниями и рисунками. Руководство
пользователя для каждой роли.
